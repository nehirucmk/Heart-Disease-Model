(this project is still under development)

**for too long, medical research has treated women as smaller versions of men. this project aims to decode the gender-based bias in cardiac diagnostics using machine learning.**

**technical stack**
language: Python
libraries: Pandas, NumPy, Scikit-learn, XGBoost
explainability: shap (Shapley Additive Explanations)
visualization: Seaborn, Matplotlib
